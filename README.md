# 坦克大战 - Battle City

经典像素风格的坦克大战网页游戏，使用 HTML5 Canvas 和原生 JavaScript 开发。

![坦克大战](screenshot.png)

## 游戏特性

- 🎮 **经典玩法**：还原 FC 版坦克大战的核心体验
- 🖥️ **像素风格**：复古像素美术风格
- 🎯 **单人模式**：玩家对抗 AI 敌人
- 📊 **关卡系统**：5个精心设计的关卡
- 🤖 **智能 AI**：敌人坦克会自动追踪和躲避
- 🔊 **音效音乐**：射击、爆炸、移动等完整音效
- 📱 **键盘控制**：WASD/方向键移动，空格射击

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格 | 射击 |
| P | 暂停游戏 |

## 敌人类型

| 类型 | 颜色 | 特点 |
|------|------|------|
| 普通型 | 红色 | 速度一般，血量1 |
| 快速型 | 蓝色 | 速度快，血量1 |
| 重型 | 橙色 | 速度慢，血量4 |

## 墙体说明

- **砖墙**（棕色）：可被子弹破坏
- **钢墙**（灰色）：不可破坏
- **草地**（绿色）：可隐藏坦克
- **水域**（蓝色）：坦克无法通过
- **基地**（黄色）：保护目标，被击中则游戏结束

## 快速开始

直接在浏览器中打开 `index.html` 文件即可开始游戏：

```bash
# 使用本地服务器（推荐）
npx serve .

# 或直接打开文件
open index.html
```

## 技术栈

- HTML5 Canvas - 游戏渲染
- 原生 JavaScript - 游戏逻辑
- Web Audio API - 音效系统
- LocalStorage - 最高分记录

## 项目结构

```
tankgame/
├── index.html          # 主页面
├── README.md           # 说明文档
└── js/
    ├── main.js         # 游戏入口
    ├── game.js         # 游戏主逻辑
    ├── entities.js     # 游戏实体（坦克、子弹等）
    ├── levels.js       # 关卡数据
    ├── ai.js           # 敌人 AI
    ├── input.js        # 输入管理
    ├── audio.js        # 音频管理
    └── utils.js        # 工具函数
```
